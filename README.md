# dApp-v2

This is dApp companion for the Silent Shard snap.

### How to install and run

- Ensure you are using Node v16+
- npm install
- npm run start

In parallel, serve the snap [Repo Link](https://github.com/silence-laboratories/silent-shard-snap.git)
